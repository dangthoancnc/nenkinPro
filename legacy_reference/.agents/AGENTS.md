Khi được yêu cầu chỉnh sửa một tính năng hoặc thêm tính năng:
1. HÃY tìm đúng dòng code cần sửa và CHỈ thay đổi đúng vị trí đó.
2. KHÔNG LÀM ẢNH HƯỞNG các mục hoặc các phần code khác đã hoạt động đúng.
3. Không viết lại (rewrite) toàn bộ file nếu không được yêu cầu rõ ràng.
